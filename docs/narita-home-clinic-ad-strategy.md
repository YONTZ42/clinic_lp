# なりたホームクリニック 採用広告・LP戦略

作成日: 2026-05-14  
対象: 看護師・准看護師の直接応募・相談獲得  
目標: 月8件の相談申し込み。Indeed・求人ボックス・LP・LINEを中心に、エージェント依存を下げる。

## 1. 前提の修正

前回整理で「オンコールなし」を強みとして扱ったが、実態が「オンコール月10件程度あり」であれば、採用上は大きな懸念点になる。

オンコールありのまま直接応募を増やすには、隠さず、候補者が判断できる情報に変える必要がある。

- 月10件程度の待機がある
- 実際の電話件数、出動件数の平均
- 医師へのエスカレーション体制
- 看護師が単独で判断する範囲
- 待機手当、電話対応手当、出動手当
- 翌日の勤務調整や代休の有無
- 未経験者がオンコールに入るまでの同行・研修期間

求人原稿では「オンコールあり」を後出しにしない。後出しにすると応募後・面談後の辞退率が上がり、媒体費が無駄になる。

## 2. 条件面の推奨

オンコール月10件がある場合、月給28万円からの表示ではかなり弱い。未経験可であっても、候補者は訪問看護・訪問診療・外来日勤と比較する。

### 最低ライン

- 正看護師: 月給30万-35万円
- 准看護師: 月給28万-32万円
- オンコール待機手当: 平日3,000円/回、休日5,000円/回
- 電話対応手当: 1,000-2,000円/件
- 出動手当: 5,000-10,000円/回
- 未経験者は入職後すぐにオンコール担当にしない

### 直接応募を取りにいく推奨ライン

- 正看護師: 月給31万-37万円
- 准看護師: 月給29万-33万円
- オンコール手当込みの想定月収を明記
- 直接応募祝い金: 10万-20万円、6ヶ月勤務後支給
- 見学・カジュアル面談・LINE相談を正式応募前に用意

### 表記例

> 月給31万円以上。オンコールは月10件程度ですが、入職後すぐの担当はありません。同行・研修後に開始し、待機手当・出動手当を別途支給します。

この表現なら、オンコールを隠さず、未経験者の不安も下げられる。

## 3. 月8件相談の逆算

成田周辺の看護師を対象にするため、母数は大きく見ない。オンコールありにより、LP・求人閲覧から相談への転換率は低めに置く。

### 悲観ケース

| 指標 | 想定 |
| --- | ---: |
| 求人・LPクリック数 | 400-800クリック/月 |
| 相談CVR | 1.0-2.0% |
| 月間相談数 | 4-16件 |
| 月8件に必要なクリック数 | 400-800クリック |

月8件を安定させるには、少なくとも月400クリック、できれば月600クリック前後を取りにいく設計が必要。

### 標準ケース

| 指標 | 想定 |
| --- | ---: |
| 求人・LPクリック数 | 250-400クリック/月 |
| 相談CVR | 2.0-3.0% |
| 月間相談数 | 5-12件 |

このケースは、写真、給与表記、オンコール説明、LINE導線が整っていることが前提。

## 4. 媒体別の役割

### Indeed

最優先。顕在層を拾う媒体。求人原稿内で「成田」「看護師」「訪問診療」「日勤」「未経験可」「オンコール手当」を明記する。

想定CPC:

- 一般的な目安: 100-300円
- 看護師・医療系で競合が強い場合: 300-600円

月予算目安:

- 最低検証: 50,000-80,000円
- 月8件を狙う運用: 100,000-180,000円

### 求人ボックス

Indeedより低CPCでクリックを取りやすい可能性がある。求人検索エンジンとして併用する。

想定CPC:

- 低め: 80-150円
- 標準: 150-300円
- 競合・上位表示狙い: 300円以上

月予算目安:

- 最低検証: 30,000-50,000円
- 月8件補助: 50,000-100,000円

### Google検索広告

LPに直接流す。検索ボリュームは小さいが、「成田 看護師 求人」「成田 訪問診療 看護師」「日勤 看護師 成田」などの高意図検索を拾える。

想定CPC:

- 地域掛け合わせ: 200-500円
- 看護師求人単体に近い語句: 500-1,200円

月予算目安:

- 最低検証: 30,000-50,000円
- 強化運用: 60,000-120,000円

Google広告はボリュームが読みづらい。Indeed・求人ボックスで取り切れない分を補う位置づけがよい。

## 5. 推奨月額予算

### LPの位置づけ

LPは、求人ボックスやIndeedに貼るだけの補足ページではなく、候補者の不安を解消してLINE相談・見学予約へ進ませる「採用の受け皿」として使う。

ただし、初期からLP単体に大きく広告費を投下しない。看護師採用では、求人検索媒体の方が顕在層に近く、成田周辺の母数も限られるため、最初はIndeed・求人ボックスを主戦場にする。

LPの主な役割:

- Indeed・求人ボックスの求人原稿では伝えきれない職場理解を補う
- オンコール月10件の実態、手当、教育体制を先に説明する
- 「応募」より手前のLINE相談・見学希望を取る
- Google検索広告の着地先にする
- 将来的に公式サイト・SNS・紹介導線からの受け皿にする

媒体別のLP導線:

| 流入元 | LPの使い方 |
| --- | --- |
| Indeed | 求人原稿内にLPまたはLINE導線を設置。応募前の詳細確認ページとして使う |
| 求人ボックス | 求人票からLPへ誘導し、写真・条件・オンコール説明で相談化する |
| Google検索広告 | LPへ直接流す。検索意図が強い語句だけに絞る |
| リスティング広告 | 初期は少額。求人媒体で拾えない検索層の補完 |
| Meta/Instagram広告 | 初期優先度は低い。潜在層向けで、相談単価が読みにくい |

結論として、LPは必須。ただし初期の集客主役は求人媒体。LP広告は検索広告を中心に少額で検証し、成果が見えたら増額する。

### 最小検証プラン

| 項目 | 月額 |
| --- | ---: |
| Indeed有料枠 | 50,000-80,000円 |
| 求人ボックス有料枠 | 30,000-50,000円 |
| Google検索広告 | 20,000-40,000円 |
| 合計 | 100,000-170,000円 |

想定相談数: 3-7件/月  
目的: 原稿・LP・LINE導線の勝ち筋確認。

配分目安:

- Indeed: 45-50%
- 求人ボックス: 30%
- Google検索広告: 20-25%

初月は求人媒体でクリックと相談の母数を作り、Google検索広告は「成田 看護師 求人」「成田 訪問診療 看護師」「看護師 日勤 成田」などに絞る。

### 月8件狙いの現実プラン

| 項目 | 月額 |
| --- | ---: |
| Indeed有料枠 | 100,000-150,000円 |
| 求人ボックス有料枠 | 50,000-80,000円 |
| Google検索広告 | 40,000-70,000円 |
| 合計 | 190,000-300,000円 |

想定相談数: 6-12件/月  
目的: 月8件の相談を狙う本命予算。

配分目安:

- Indeed: 50%
- 求人ボックス: 25-30%
- Google検索広告: 20-25%

月8件を狙う段階では、LP広告にも一定額を入れる。ただしLP広告は広げすぎず、検索広告中心にする。求人媒体で反応のよい訴求を、Google広告の見出しにも転用する。

### 強化プラン

| 項目 | 月額 |
| --- | ---: |
| Indeed有料枠 | 150,000-220,000円 |
| 求人ボックス有料枠 | 80,000-120,000円 |
| Google検索広告 | 70,000-120,000円 |
| 合計 | 300,000-460,000円 |

想定相談数: 9-18件/月  
目的: 短期で母数を作り、エージェント比率を明確に下げる。

配分目安:

- Indeed: 45-50%
- 求人ボックス: 25-30%
- Google検索広告: 20-25%
- 必要に応じてMeta/Instagram: 0-10%

強化プランでも、Meta/Instagramは最初から主戦場にしない。成田周辺の看護師に絞ると配信母数が小さく、潜在層向け広告は相談化まで時間がかかるため。

### LP広告を増やす判断基準

LPへのGoogle検索広告は、以下を満たす場合に増額する。

- LP相談CVRが2.0%以上
- LINE追加後の相談化率が50%以上
- オンコール説明後の離脱が許容範囲
- 1相談あたり広告費が25,000円以内

逆に、LP相談CVRが1.0%未満なら、広告費を増やす前にファーストビュー、給与表記、オンコール説明、LINE導線を修正する。

### 初期の推奨配分

初月の推奨は、月20万円前後。

| 媒体 | 月額 | 役割 |
| --- | ---: | --- |
| Indeed | 100,000円 | 顕在層の主力。求人検索中の看護師を取る |
| 求人ボックス | 60,000円 | Indeedの補完。クリック単価を見ながら調整 |
| Google検索広告 | 40,000円 | LP直接流入。高意図検索だけ拾う |
| 合計 | 200,000円 | 月8件の下限を検証 |

2ヶ月目以降は、相談単価で配分を変える。

| 状況 | 次の配分 |
| --- | --- |
| Indeedの相談単価が低い | Indeedへ追加配分 |
| 求人ボックスのクリックは安いが相談が少ない | LP・原稿を修正し、予算は据え置き |
| Google検索広告のCVRが高い | Googleを月7万-10万円まで増額 |
| LP流入はあるがLINE追加が少ない | 広告増額せずLPを改善 |
| オンコール説明後の離脱が多い | 手当・開始時期・出動頻度の見せ方を改善 |

## 6. 予算判断

エージェント経由で看護師を1名採用すると、年収の20-35%程度の紹介料が発生することが多い。年収450万円なら90万-157.5万円程度になる。

そのため、3ヶ月で60万-90万円の広告費を使って1名採用できるなら、エージェント依存を下げる投資として成立しやすい。

ただし、オンコール月10件がある場合、広告費だけで解決しない。条件表記と手当設計を変えないと、クリックは取れても相談率・面談率が落ちる。

## 7. LP訴求パターン

### パターンA: 未経験から訪問診療

主訴:

> はじめての在宅医療は、医師と一緒に訪問するクリニックで。

向いている候補者:

- 病棟・外来から在宅医療に移りたい
- 訪問看護の単独訪問に不安がある
- 未経験でも教育がある職場を探している

ファーストビュー要素:

- 訪問未経験可
- 医師と同行
- 入職後すぐのオンコールなし
- 月給31万円以上
- 見学だけOK

オンコールの伝え方:

> オンコールは月10件程度。未経験の方は業務に慣れてから担当します。対応範囲や手当は面談前にLINEで確認できます。

このパターンは相談CVRを最も取りやすい。月8件相談の主軸にする。

### パターンB: 夜勤から日勤常勤へ

主訴:

> 夜勤を離れて、成田で日勤中心の看護を続ける。

向いている候補者:

- 病棟夜勤から離れたい
- 子育て・家庭と両立したい
- 夜勤は避けたいが、看護師としての医療行為は続けたい

ファーストビュー要素:

- 8:30-17:30
- 水日休み
- 年間休日120日
- 夜勤なし
- オンコール手当別途支給

オンコールの伝え方:

> 夜勤はありません。オンコールは月10件程度ありますが、待機・出動には手当を支給し、対応体制を明確にしています。

このパターンは「オンコールなし」を求める層が離脱するため、CVRはやや下がる。ただし夜勤離脱ニーズには刺さる。

### パターンC: 地元で長く働く在宅医療

主訴:

> 成田で、地域の暮らしを支える訪問診療看護。

向いている候補者:

- 成田・富里・酒々井周辺で働きたい
- 車通勤したい
- 大病院より顔が見える職場で働きたい
- 地域医療に関心がある

ファーストビュー要素:

- 成田赤十字病院敷地内
- 車通勤可
- 訪問診療クリニック
- 少人数で相談しやすい
- LINEで職場見学予約

オンコールの伝え方:

> 少人数のためオンコール分担はあります。月10件程度を想定し、担当開始時期・手当・出動頻度を事前に説明します。

このパターンは応募数は伸びにくいが、定着期待の高い候補者に合いやすい。

## 8. LP構成

LPは1ページでよい。媒体別にファーストビューと見出しだけ差し替えられる設計にする。

1. ファーストビュー
   - 実際のスタッフ写真
   - 月給、勤務時間、休日、オンコール手当
   - CTA: LINEで質問、見学予約

2. 仕事内容
   - 訪問診療同行
   - 外来補助
   - 採血・検査補助
   - 記録・物品管理
   - 多職種連携

3. 未経験者への教育
   - 同行期間
   - オンコール開始時期
   - 判断に迷った時の相談先

4. オンコールの実態
   - 月10件程度
   - 平均電話件数
   - 平均出動件数
   - 手当
   - 医師との役割分担

5. 1日の流れ
   - 8:30出勤
   - 午前訪問
   - 昼休憩
   - 午後訪問
   - 記録・物品整理
   - 17:30退勤

6. 職場の雰囲気
   - 院内写真
   - 診療車
   - 休憩スペース
   - スタッフ写真

7. 条件表
   - 給与
   - 賞与
   - 昇給
   - 休日
   - オンコール
   - 手当
   - 試用期間

8. LINE相談
   - 条件だけ知りたい
   - 見学したい
   - 未経験なので相談したい
   - 今すぐではないが話を聞きたい

## 9. 広告原稿の方向性

### Indeed・求人ボックス用タイトル案

- 訪問診療クリニックの看護師｜未経験可｜月給31万円以上
- 成田市の訪問診療看護師｜医師同行｜夜勤なし
- 看護師・准看護師｜訪問未経験可｜見学だけOK

### Google検索広告の広告文案

広告見出し:

- 成田市の訪問診療看護師
- 未経験可・医師同行
- LINEで見学相談OK
- 夜勤なし・月給31万円以上

説明文:

- 成田市の訪問診療クリニック。医師同行で未経験でも始めやすい看護業務。オンコール体制・手当も事前に確認できます。
- 正式応募前にLINE相談・職場見学が可能。仕事内容、1日の流れ、給与、オンコールの実態を公開しています。

## 10. 運用KPI

| 指標 | 最低ライン | 目標 |
| --- | ---: | ---: |
| 求人クリック率 | 1.0% | 1.5%以上 |
| LP相談CVR | 1.5% | 3.0%以上 |
| LINE追加から相談化 | 50% | 70%以上 |
| 相談から見学・面談 | 40% | 60%以上 |
| 月間相談数 | 5件 | 8件以上 |

週次で見る指標:

- 面談・見学数

## 11. 初月の運用方針

初月から広告費を絞りすぎると、学習に必要なクリック数が足りない。最初の4週間は最低でも10万-17万円、可能なら19万-30万円で検証する。

初月に判断すること:

- クリックは取れているか
- オンコールありでLINE追加が落ちていないか
- 給与表記で離脱していないか
- 未経験可の訴求が反応しているか
- 見学希望と条件確認のどちらが多いか

勝ち筋が見えたら、2ヶ月目以降は反応のよい媒体・訴求に予算を寄せる。
