# なりたホームクリニック 採用戦略メモ

作成日: 2026-05-14  
対象: 看護師・准看護師の直接応募獲得  
目標: 月8件の相談申し込み獲得。可能な限りIndeed・求人ボックス・自社導線で獲得し、エージェント依存を減らす。

## 1. クリニックの現状

### 基本情報

- 医療機関名: なりたホームクリニック
- 所在地: 千葉県成田市飯田町90番地1 成田赤十字病院アメニティ棟1階
- 診療内容: 訪問診療、内科、泌尿器科、精神科
- 訪問診療の診療日: 月・火・木・金・土
- 休診日: 水・日
- 掲げている特徴: 24時間365日対応、在宅医療、地域連携

### 採用上の現状課題

- 公式ホームページは診療情報中心で、採用候補者が知りたい「職場の雰囲気」「スタッフ」「教育体制」「1日の流れ」が弱い。
- 院内・訪問同行・診療車・休憩スペースなどの写真が不足しており、働くイメージを持ちにくい。
- 求人媒体では「月給28万円から」が先に見え、周辺の高給与求人と比較されたときに弱く見える。
- 訪問診療同行の安心感、17:30終業、年間休日120日、オンコール手当・教育体制などの強みが十分に前面化されていない。
- エージェント経由を減らすには、応募より手前の「相談」「見学」「カジュアル面談」を拾う導線が必要。

### 採用上の強み

- 夜勤なし
- オンコールは月10件程度。手当、実際の電話・出動頻度、未経験者の開始時期を明示できれば不安を下げられる
- 8:30-17:30勤務
- 年間休日120日
- 完全週休2日制
- 水日休みで予定を固定しやすい
- 成田赤十字病院敷地内という立地の安心感
- 訪問看護ではなく、医師と同行する訪問診療である点
- 訪問未経験者でも入りやすい可能性がある

### 採用上の弱み

- 給与レンジの見え方が競合より弱い。
- 少人数運営のため、候補者からは「忙しそう」「休みにくそう」と受け取られる可能性がある。
- ホームページ上でスタッフの人柄や院内の雰囲気が見えにくい。
- 訪問診療の仕事内容が具体化されておらず、訪問看護との違いが伝わりにくい。

## 2. 周辺環境と採用市場

### 対象エリア

主対象は成田市内および周辺地域の看護師。

- 成田市
- 富里市
- 酒々井町
- 栄町
- 印西市の一部
- 佐倉市の一部
- 芝山町・多古町方面

車通勤が前提になりやすいため、求人訴求では「駐車場」「通勤しやすさ」「訪問エリア」「運転頻度」を明記する。

### 候補者像

優先すべき候補者は以下。

- 病棟夜勤から日勤常勤へ移りたい看護師
- 夜勤を避けたいが、条件と体制が明確ならオンコール月10件程度は検討できる看護師
- 訪問看護の単独訪問には不安があるが、在宅医療には関心がある看護師
- ブランク後に医療現場へ戻りたい看護師
- 成田周辺で長く働きたい地元志向の看護師
- クリニック外来だけではなく、地域医療に関わりたい看護師

### 悲観的な市場見込み

成田市周辺だけで看護師採用を行う場合、母数は大きくない。月8件の相談獲得は可能性があるが、条件改善・LP改善・広告運用の組み合わせが前提。

悲観ケースの目安:

| 指標 | 月間見込み |
| --- | ---: |
| Indeed・求人ボックス広告表示 | 8,000-15,000回 |
| 求人クリック率 | 1.0-1.5% |
| LP流入 | 80-225件 |
| LINE追加・相談率 | 3-5% |
| 相談申し込み | 2-11件 |

現条件の見せ方のままでは月8件は厳しい。月8件を狙うには、求人条件の見直し、写真、LP、LINE導線、広告運用を同時に改善する必要がある。

## 3. 周辺求人の相場感

公開求人から見た周辺相場の要約。

| 事業所・求人タイプ | 条件の見え方 |
| --- | --- |
| 訪問診療クリニック | 月給30万円前後、土日祝休み、年間休日120日以上が競合要素 |
| 訪問看護ステーション | 月給32万-40万円以上の表示が多く、給与面では強い |
| クリニック外来 | 月給26万-32万円前後。夜勤なし・日勤固定が訴求軸 |
| パート看護師 | 時給1,800-2,200円前後が比較対象 |

### 競合と比較したときの課題

- 月給28万円からの表示は、訪問看護の月給32万-40万円求人に対して弱く見える。
- 訪問診療クリニック同士でも、月給30万円、賞与4ヶ月、土日祝休み、年間休日129日などが見えると相対的に不利。
- ただし、訪問看護の高給与求人はオンコール、単独訪問、業務負荷がある場合も多い。なりたホームクリニックは「医師同行」「未経験者への教育」「17:30終業」「オンコール手当と実態の明示」を明確にすれば別軸で勝てる。

## 4. あるべき条件面

### 標準案

既存条件を大きく崩さず、競合比較で弱く見えない最低ライン。

- 正看護師: 月給30万-35万円
- 准看護師: 月給28万-32万円
- パート正看護師: 時給2,000-2,200円
- パート准看護師: 時給1,800-2,000円
- 年間休日120日
- 8:30-17:30
- オンコール月10件程度。待機手当・出動手当を別途支給
- 完全週休2日制

求人表記では「月給30万円以上可」「17:30終業」「オンコール手当あり」「未経験者は慣れてからオンコール開始」を先に見せる。

### 直接応募強化案

エージェント費用を削減する目的で、直接応募者に還元する設計。

- 正看護師: 月給31万-36万円
- 准看護師: 月給29万-33万円
- 直接応募祝い金: 10万-20万円
- 支給条件: 入職6ヶ月後に支給
- 見学・カジュアル面談OK

エージェント経由の紹介料と比較すると、直接応募祝い金は十分に合理的。媒体原稿・LP・LINEで明記する。

### 即戦力採用案

訪問診療・訪問看護・地域連携経験者を取りたい場合。

- 訪問診療または訪問看護経験者: 月給33万-38万円
- 役割手当: 月1万-3万円
- リーダー候補手当: 月2万-5万円
- 運転可、在宅医療経験、多職種連携経験を評価

「経験者は前職給与考慮」だけでは弱い。経験者用の給与レンジを明記する。

## 5. 採用戦略

### 基本方針

求人媒体で条件を見せ、LPで職場理解を深め、LINEで相談化する。

応募完了を最初のゴールにしない。成田周辺では母数が限られるため、「今すぐ転職ではないが興味がある」層をLINEで拾い、見学・面談へ育成する。

### 訴求軸

主訴:

> 夜勤なし。医師と同行しながら、未経験から在宅医療に関われる成田のクリニック

補助訴求:

- 医師と一緒に訪問するため、訪問未経験でも始めやすい
- 17:30終業で生活リズムを崩しにくい
- 年間休日120日
- 成田赤十字病院敷地内で通勤しやすい
- 見学だけでも歓迎
- 履歴書なしでLINE相談可能

### 求人原稿の方向性

求人媒体では、冒頭3行で強みを伝える。

例:

> 夜勤なし・17:30終業の訪問診療クリニックです。  
> 医師と一緒に患者さま宅へ訪問するため、訪問未経験の方も始めやすい仕事です。  
> 見学・LINE相談からでも歓迎しています。

避けたい表現:

- アットホーム
- やりがいがあります
- 少人数で頑張っています
- 未経験歓迎だけを強調する

少人数運営は強みにも弱みにもなるため、「相談しやすい」「顔が見える」「業務範囲が明確」とセットで伝える。

## 6. LP戦略

### LPの目的

求人媒体から流入した候補者に、条件だけでは伝わらない安心材料を与え、LINE相談・見学・カジュアル面談へ進ませる。

### 推奨ページ構成

1. ファーストビュー
   - スタッフ写真、診療車、院内写真を使用
   - 月給30万円以上、17:30終業、オンコール手当、年休120日、見学OKを表示
   - CTAは「LINEで質問する」「見学を申し込む」

2. 仕事内容
   - 訪問診療同行
   - 外来補助
   - 採血・検査補助
   - 物品管理
   - 電話対応・多職種連携
   - 訪問看護との違いを説明

3. 1日の流れ
   - 8:30 出勤
   - 午前 訪問診療同行
   - 昼休憩
   - 午後 訪問診療同行
   - 夕方 記録・物品整理
   - 17:30 退勤

4. 職場の雰囲気
   - スタッフ人数
   - 年齢層
   - 院内写真
   - 休憩スペース
   - 教育・同行期間

5. 条件表
   - 給与
   - 賞与
   - 昇給
   - 休日
   - 残業
   - オンコール
   - 試用期間
   - 車通勤・駐車場

6. よくある不安
   - 訪問未経験でも大丈夫か
   - 運転は必要か
   - 訪問エリア
   - 残業
   - 子育てとの両立
   - ブランク
   - 准看護師応募可否

7. 申し込み
   - LINE相談
   - 見学申し込み
   - 15分カジュアル面談

### LPに必要な素材

- 院内写真
- 外観写真
- 診療車の写真
- 訪問バッグ・物品の写真
- スタッフ集合写真
- 院長または採用担当の写真
- 実際の制服姿
- 休憩スペース
- 地図・駐車場案内

## 7. LINE導線

### 目的

履歴書提出や正式応募の前に、心理的ハードルの低い相談導線を作る。

### 初回メニュー

- 条件を詳しく知りたい
- 職場見学したい
- 訪問診療未経験なので相談したい
- 今すぐではないが話を聞きたい

### 自動返信で送る情報

1. 勤務条件の要約
2. 給与・休日・オンコール有無
3. 仕事内容
4. 院内・スタッフ写真
5. 1日の流れ
6. 見学・面談予約URL

### 入力項目

最初から電話番号や履歴書を求めない。初回は以下で十分。

- 名前
- 看護師または准看護師
- 希望: 見学、相談、応募、情報収集
- 希望連絡方法
- 希望日時

電話番号は面談予約時点で取得する。

## 8. 広告戦略

### 媒体優先度

1. Indeed
2. 求人ボックス
3. Google検索広告
4. Meta広告
5. 公式サイト・LPのSEO

Indeed・求人ボックスは顕在層向け。Google検索広告は「成田 看護師 日勤」「成田 訪問診療 看護師」などの検索を拾う。Meta広告は潜在層向けで、LINE追加や見学誘導に使う。

### 広告訴求パターン

#### パターンA: 夜勤なし訴求

- 夜勤なし
- オンコール手当あり
- 17:30終業
- 年間休日120日

#### パターンB: 訪問未経験訴求

- 医師と同行する訪問診療
- 一人訪問ではない
- 在宅医療を学べる
- 見学からOK

#### パターンC: 地元勤務訴求

- 成田で日勤常勤
- 車通勤可
- 成田赤十字病院敷地内
- 地域医療に関われる

### 運用KPI

| 指標 | 目安 |
| --- | ---: |
| 求人クリック率 | 1.0%以上 |
| LPからLINE追加率 | 3%以上 |
| LINE追加から相談化率 | 40%以上 |
| 相談から見学・面談化率 | 50%以上 |
| 相談申し込み | 月8件 |

### 月8件に必要な逆算

保守的に見ると、以下が必要。

- LP訪問: 月200件
- LINE追加率: 4%
- LINE追加: 月8件
- 相談化率: 70-100%

または、

- LP訪問: 月300件
- LINE追加率: 3%
- LINE追加: 月9件

成田周辺では流入数が伸びにくい可能性があるため、LP訪問数よりもLINE追加率を高める設計が重要。

## 9. 実行優先順位

### Phase 1: すぐ実施

- 採用LPを作る
- LINE相談導線を作る
- 求人原稿を3パターン作る
- 直接応募祝い金の可否を決める
- 給与表記を月給30万円以上に見える形へ調整する
- 写真撮影を行う

### Phase 2: 運用開始後

- Indeed・求人ボックスで求人原稿をABテスト
- LPのファーストビューとCTAを改善
- LINE追加後の自動返信を改善
- 相談後の見学率を確認
- 条件面で離脱している場合は給与・休日・祝い金を再調整

### Phase 3: 中期

- 公式ホームページにも採用導線を追加
- スタッフインタビューを追加
- 訪問診療未経験者向けの教育ページを追加
- エージェント経由応募の比率を下げる

## 10. 提案時の言い方

ホームページ全面改修が通りにくい場合は、採用LPを先に提案する。

提案例:

> 公式サイト全体を作り替えるより先に、採用専用LPとLINE相談導線を作るのが費用対効果として現実的です。現在の求人は条件面だけで比較されやすく、職場の雰囲気や訪問診療同行の安心感が伝わっていません。LPで写真・仕事内容・1日の流れ・条件表を見せ、LINEで見学や相談を受けることで、エージェントに頼らない直接応募の母数を増やします。
