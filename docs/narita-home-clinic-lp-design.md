# なりたホームクリニック 採用LP設計

作成日: 2026-05-15  
対象: 看護師・准看護師の直接応募・LINE相談・職場見学獲得  
目的: 既存の採用戦略・広告LP戦略をもとに、AIっぽく見えない、実在感のある採用LPを設計する。

## 1. LPの基本方針

このLPは、求人媒体で条件を見た候補者に対して、応募前の不安を減らすための採用専用ページにする。

主な役割:

- 「訪問診療」と「訪問看護」の違いを直感的に伝える
- 医師同行、教育、オンコール体制を隠さず説明する
- 給与・勤務時間・休日をファーストビューで比較しやすく見せる
- 正式応募ではなく、LINE相談・見学・15分面談へ進ませる
- 写真と現場感で「本当にここで働く」イメージを作る

メイン訴求:

> はじめての在宅医療は、医師と一緒に訪問するクリニックで。

補助訴求:

- 夜勤なし、8:30-17:30
- 医師同行の訪問診療
- 訪問未経験でも同行から開始
- 月給31万円以上を目指す表記
- オンコールは月10件程度。手当と開始時期を事前に説明
- 成田赤十字病院敷地内
- LINEで質問、見学だけでも可

## 2. 参考サイトから取り入れる要素

参照したクリニック系サイト・記事:

- けやき在宅クリニック採用サイト: https://soshokai.jp/
- ゆう在宅クリニック採用サイト: https://you-ohshin.biz/lp/
- Acspor Medical「見やすいクリニックホームページデザイン事例9選」: https://medical.acsport.net/column/column-2615/
- ヘッドスプリングDMS クリニック制作事例LP: https://dms.head-spring.co.jp/lp/homepage2602/
- 厚生労働省 医療広告規制におけるウェブサイトの事例解説書: https://www.mhlw.go.jp/content/10800000/001153837.pdf

取り入れるポイント:

- 在宅医療サイトは、診療車・訪問バッグ・患者宅前・スタッフの移動シーンを見せると仕事内容が伝わりやすい。
- 採用LPでは、給与や条件だけでなく「働く理由」「教育」「1日の流れ」「先輩の声」を写真付きで並べると相談化しやすい。
- 直接応募祝い金やWEB応募特典は、エージェント依存を下げる導線として有効。ただし条件は明確にする。
- 高級感を出す場合も、抽象的なグラデーションではなく、写真の色調、余白、文字組み、紙面のようなレイアウトで作る。
- 医療広告では、体験談・効果保証・誇大表現・ビフォーアフターの扱いに注意する。採用LPでも「必ず」「誰でも」などの断定は避ける。

## 3. AIっぽさを消すデザイン原則

避けるもの:

- 丸すぎるカード、角丸24px以上の汎用UI
- 青紫のグラデーション、淡いパステルだけの配色
- 影付きカードを等間隔に並べるだけの構成
- 生成AI風の完璧すぎる笑顔、均一すぎる肌、清潔すぎる無人の診察室
- Noto Sansだけで全体を作る「普通のWebテンプレート感」
- 抽象イラスト、医療アイコンの大量使用、丸い資格バッジ
- 「地域に寄り添う」「安心の医療」だけで終わる薄いコピー

採用するもの:

- 写真は横長、縦長、寄り、引きを混ぜる。全て同じ比率にしない
- 角丸は0-8px。写真は角丸なし、情報ボックスのみ4px程度
- 白地を基本に、黒に近い墨色、深い緑、赤土色、生成りを使う
- 見出しは明朝体またはコントラストのあるセリフ体を一部に使う
- 本文は可読性重視のゴシック。数字・条件はやや詰まったサンセリフ
- 余白は大きく取りすぎず、採用情報らしい密度を残す
- 写真の粒子、窓光、紙の質感、現場の小物を残す
- CTAは大きい丸ボタンではなく、直線的な帯・固定下部バー・電話番号風の現実的な形にする

推奨トーン:

- 清潔だが無菌すぎない
- 温かいが可愛すぎない
- 地域感はあるが田舎っぽくしない
- 採用サイトらしく、条件と現場感を強く見せる

## 4. ビジュアルコンセプト

コンセプト名:

> 成田で、日々の暮らしへ向かう看護。

ビジュアルの核:

- 朝のクリニック前、診療車、訪問バッグ、制服姿のスタッフ
- 医師と看護師が訪問前にカルテ・物品を確認する場面
- 成田の生活道路、住宅街、低い空、赤十字病院周辺の安心感
- 院内の手元、記録、物品棚、バッグの中身、スタッフ同士の相談

写真の方向性:

- モデル写真ではなく、実在の採用ドキュメンタリーのように見せる
- カメラ目線の集合写真は1枚だけ。基本は仕事中の横顔・手元・移動
- 逆光や自然光を使う。真っ白な医療広告写真にしない
- 色温度はやや暖かめ。緑と赤土色を少し入れ、単調な青白さを避ける

## 5. ページ構成

### 5.1 ファーストビュー

目的:

求人媒体から来た候補者に、3秒で「自分向けかも」と判断させる。

構成:

- 左側: キャッチコピー、条件、CTA
- 右側または背景: 訪問前に診療車へ向かう医師・看護師の写真
- 下部: 条件の細い帯

コピー案:

> はじめての在宅医療は、  
> 医師と一緒に訪問するクリニックで。

サブコピー:

> 成田市の訪問診療クリニックで、看護師・准看護師を募集しています。夜勤なし。8:30-17:30。訪問未経験の方も、同行と相談体制から始められます。

条件帯:

| 表示 | 内容 |
| --- | --- |
| 勤務 | 8:30-17:30 / 夜勤なし |
| 休日 | 年間休日120日 / 水日休み |
| 給与 | 正看護師 月給31万円以上目安 |
| 訪問 | 医師同行 / 単独訪問ではない |
| 相談 | LINEで質問・見学OK |

CTA:

- 主: LINEで条件を質問する
- 副: 見学を申し込む

注意:

- 「オンコールなし」に見える表現は避ける。
- ファーストビュー内に小さく「オンコール月10件程度。手当・開始時期は事前に説明」と表示する。

### 5.2 こんな方へ

目的:

候補者の悩みとLPの訴求を接続する。

見出し:

> 病棟から日勤へ。外来から在宅へ。  
> でも、一人で訪問するのは不安な方へ。

内容:

- 夜勤を続ける働き方を見直したい
- 訪問看護の単独判断には不安がある
- 成田周辺で長く働きたい
- ブランク後に少しずつ戻りたい
- 患者さんの生活に近い看護に関わりたい

ビジュアル:

- 余白のあるテキストカードではなく、左に現場写真、右に短い箇条書き
- 写真は休憩中の会話、訪問バッグの準備、カルテ確認

### 5.3 仕事内容

目的:

訪問診療同行の実務を具体化し、訪問看護との違いを伝える。

見出し:

> 医師と一緒に訪問し、診療を支える看護です。

内容:

- 診療車で患者さま宅へ移動
- 医師の診療補助
- バイタル測定、採血、検査補助
- 記録、物品管理、訪問準備
- 患者さま・ご家族・関係機関との連絡
- 外来補助、電話対応

訪問看護との違い:

| 訪問診療同行 | 訪問看護 |
| --- | --- |
| 医師と一緒に患者宅へ訪問 | 看護師が単独訪問する場面が多い |
| 診療補助・検査補助・連携が中心 | ケア計画に基づく看護実践が中心 |
| 判断に迷った時に医師へ相談しやすい | 現場での単独判断が必要な場面がある |

ビジュアル:

- 診療車のドアを開ける手元
- 訪問バッグの中身
- 医師と看護師がタブレットを見る場面

### 5.4 未経験者への教育

目的:

「訪問未経験でも本当に大丈夫か」を解消する。

見出し:

> 最初から一人で抱え込ませないために。

内容:

1. 入職初週: 院内業務、物品、記録、訪問の流れを確認
2. 2週目以降: 医師・先輩と訪問同行
3. 慣れてから: 電話対応、関係機関連絡を段階的に担当
4. オンコール: 業務理解後に開始時期を相談

CTA前の補足:

> 教育期間やオンコール開始時期は、経験やブランクに応じて面談時に相談できます。

### 5.5 オンコールの実態

目的:

候補者が最も気にする懸念を、後出しにせず信頼材料に変える。

見出し:

> オンコールはあります。だから、事前にきちんと説明します。

掲載項目:

- 月10件程度の待機想定
- 実際の電話件数、出動件数の平均
- 看護師が判断する範囲
- 医師へのエスカレーション体制
- 待機手当、電話対応手当、出動手当
- 未経験者の開始時期
- 翌日の勤務調整の考え方

表現例:

> オンコールは月10件程度を想定しています。入職後すぐに担当するのではなく、訪問の流れや対応範囲を理解してから開始します。待機・電話対応・出動には手当を支給します。

避ける表現:

- 負担はほとんどありません
- すぐ慣れます
- 安心してください
- 家庭と必ず両立できます

### 5.6 1日の流れ

目的:

勤務後の生活リズムを想像させる。

構成:

| 時間 | 内容 |
| --- | --- |
| 8:30 | 出勤、訪問準備、申し送り |
| 9:00 | 午前の訪問診療同行 |
| 12:00 | 休憩 |
| 13:00 | 午後の訪問診療同行 |
| 16:30 | 記録、物品整理、翌日の準備 |
| 17:30 | 退勤 |

ビジュアル:

- 横長の時系列ではなく、縦スクロールで写真を挟む
- 朝、昼、夕方で光の色を変え、1日の実感を出す

### 5.7 職場の雰囲気

目的:

少人数運営の不安を「顔が見える」「相談しやすい」に変える。

掲載内容:

- スタッフ人数
- 年齢層
- 院内・休憩スペース
- 診療車
- 相談の仕方
- 入職後に聞ける相手

写真:

- 集合写真は整列しすぎない
- 診療車の前で自然に会話している写真
- 院内の小物、掲示、物品棚など現実味のあるカット

### 5.8 条件表

目的:

求人媒体との比較で離脱させない。

掲載項目:

| 項目 | 表示案 |
| --- | --- |
| 職種 | 看護師・准看護師 |
| 雇用形態 | 正社員、必要に応じてパート |
| 給与 | 正看護師 月給31万-37万円目安 / 准看護師 月給29万-33万円目安 |
| 勤務時間 | 8:30-17:30 |
| 休日 | 完全週休2日制、水曜・日曜、年間休日120日 |
| オンコール | 月10件程度。手当別途支給 |
| 勤務地 | 千葉県成田市飯田町90番地1 成田赤十字病院アメニティ棟1階 |
| 通勤 | 車通勤・駐車場の有無を明記 |
| 応募前相談 | LINE相談、見学、15分面談 |

### 5.9 FAQ

目的:

LINE相談前の不安を減らす。

質問:

- 訪問診療が未経験でも応募できますか
- 訪問看護との違いは何ですか
- 運転は必要ですか
- オンコールはいつから担当しますか
- 実際の出動頻度はどれくらいですか
- ブランクがあっても相談できますか
- 准看護師も応募できますか
- 見学だけでも可能ですか

### 5.10 最終CTA

目的:

正式応募の前段階を拾う。

見出し:

> 応募する前に、条件だけ聞いてみることもできます。

ボタン:

- LINEで質問する
- 見学を申し込む
- 15分だけ話を聞く

入力項目:

- 名前
- 看護師 / 准看護師
- 希望: 見学、相談、応募、情報収集
- 希望連絡方法
- 希望日時

初回で履歴書・職務経歴書は求めない。

## 6. デザイン仕様

### 色

AIっぽい青紫グラデーションを避け、写真に合う現実的な色にする。

| 用途 | 色 | メモ |
| --- | --- | --- |
| 背景 | #F7F4EE | 生成り。真っ白を避ける |
| 本文 | #202020 | 墨色 |
| 見出し | #14231F | 黒に近い深緑 |
| アクセント | #8B3F2F | 赤土色。CTAや数字に使う |
| サブアクセント | #3D6658 | 医療感のある深緑 |
| 罫線 | #D8D0C5 | 紙面に近い薄い線 |
| 注意 | #7A2E24 | オンコール説明など |

### フォント

- 見出し: 游明朝、Yu Mincho、Shippori Mincho、またはNoto Serif JP
- 本文: BIZ UDPGothic、Yu Gothic、Hiragino Sans
- 数字・条件: Inter、Arial、またはDIN系

文字組み:

- 見出しは字間を広げすぎない
- 英字の装飾見出しは小さく使う
- ボタン文言は太字にしすぎない
- 行間は本文1.8、条件表1.5程度

### レイアウト

- PC幅: 最大1120px
- 余白: セクション上下72-96px
- スマホ: CTAを下部固定。電話番号風の現実的なバーにする
- カード角丸: 0-8px
- 写真角丸: 基本0px
- ボタン角丸: 2-4px
- 影: 原則使わない。罫線・背景差・写真で分ける

## 7. 必要な写真・画像

実写撮影を優先するもの:

- 外観、成田赤十字病院アメニティ棟の入口
- 診療車
- 訪問バッグ、物品
- 医師と看護師の訪問前ミーティング
- タブレットやカルテ確認
- 休憩スペース
- スタッフ集合写真
- 院長または採用担当者

生成画像で補完してよいもの:

- ファーストビューの空気感を作るキービジュアル
- 訪問バッグ・診療車・住宅街のイメージ
- セクション間の背景写真
- 広告バナー用のビジュアル

生成画像で避けるもの:

- 実在スタッフの顔の代替
- 医療行為中の正確性が必要な場面
- 患者の顔がはっきり写る場面
- 成田赤十字病院の建物を事実と異なる形で描くこと

## 8. GPTimage2.0用 画像構成とプロンプト

前提:

- 実装時にGPTimage2.0が使える場合、下記プロンプトで画像を生成する。
- すべて「AI生成っぽさ」を避けるため、広告写真ではなくドキュメンタリー写真の指示にする。
- 人物の顔は完璧に整えず、表情は控えめにする。
- 医療行為の細部は描かず、訪問前後の準備・移動・相談を中心にする。

### 8.1 ファーストビュー

用途:

- LP最上部のメイン画像
- PCでは右側大判、スマホでは上部背景

構図:

- 朝の自然光
- 診療車の横で、医師と看護師が訪問バッグを持って出発する
- 看護師は30-40代の日本人女性、医師は白衣ではなく落ち着いた診療着
- 顔は横顔または後ろ姿寄り
- 背景は千葉県成田市周辺の現実的な住宅街・医療施設周辺の雰囲気

プロンプト:

```text
Documentary-style recruitment photo for a small home medical clinic in Narita, Chiba, Japan. Early morning natural light, a doctor and a nurse preparing to leave beside a compact clinic car, carrying a practical home-visit medical bag. The nurse is a Japanese woman in her 30s or 40s wearing simple navy medical scrubs, the doctor wears understated clinical workwear, not a dramatic white coat. Realistic Japanese suburban clinic surroundings, quiet road, muted signage, practical atmosphere. Shot like an editorial photograph for a healthcare recruitment magazine, 35mm lens, natural skin texture, imperfect real-world details, subtle film grain, restrained colors, deep green and warm off-white tones. No futuristic elements, no glossy tech look, no gradient background, no smiling stock-photo poses, no AI-like perfect faces, no exaggerated medical procedure, no patient visible.
```

ネガティブ要素:

```text
avoid: futuristic hospital, blue purple gradient, rounded app-like badges, perfect symmetrical faces, plastic skin, overly bright white room, stock photo smile, luxury clinic, anime style, illustration, fake Japanese text, visible patient face, surgical scene
```

### 8.2 訪問前ミーティング

用途:

- 仕事内容セクション
- 医師同行の安心感を表現

構図:

- 院内の小さなテーブル
- 医師と看護師がタブレット・訪問バッグ・紙のチェックリストを見ている
- 手元中心、顔は一部だけ

プロンプト:

```text
Realistic editorial photo inside a modest Japanese home-visit clinic office. A doctor and a nurse review the day schedule together at a small table with a tablet, paper checklist, pens, blood pressure monitor, and a packed home-visit medical bag. Focus on hands, documents, and practical tools, faces partially visible and natural. Warm window light, compact real clinic workspace, slightly lived-in details, clean but not sterile. Shot for a nursing recruitment landing page, documentary style, 50mm lens, natural colors, subtle shadows, no staged corporate look.
```

ネガティブ要素:

```text
avoid: oversized modern hospital, glowing holograms, generic laptop mockup, perfect white background, round icon overlays, stock photo handshake, fake readable text, exaggerated smiles, luxury interior
```

### 8.3 訪問バッグと物品

用途:

- 仕事内容、教育、1日の流れの間に挿入
- 仕事の具体性を出す

構図:

- 訪問バッグの中身を斜め上から
- 聴診器、血圧計、手袋、アルコール、書類、ペン
- 人の手が一部入る

プロンプト:

```text
Close-up documentary photograph of a practical home medical visit bag being prepared in a Japanese clinic. Items include a blood pressure monitor, stethoscope, disposable gloves, alcohol wipes, documents, pens, small medical supplies, neatly organized but used in real work. A nurse's hands are checking the contents. Warm off-white desk, deep green fabric accent, realistic texture, soft side light, 45-degree overhead composition, editorial healthcare recruitment style, subtle film grain.
```

ネガティブ要素:

```text
avoid: sterile product catalog, floating objects, glowing icons, unrealistic medical devices, fake labels, overly perfect arrangement, blue gradient, cartoon style
```

### 8.4 夕方の記録・退勤前

用途:

- 1日の流れ、17:30退勤訴求
- 夜勤なしの生活感を表現

構図:

- 夕方の院内
- 看護師が記録を終えてバッグを閉じる
- 窓から夕方の光
- 時計は17:20前後を示す。ただし文字盤が破綻する可能性があるため、時計はぼかす

プロンプト:

```text
Editorial documentary photo in a small Japanese clinic office at late afternoon. A nurse finishes documentation and closes a home-visit medical bag before leaving work. Warm low sunlight from a window, quiet end-of-day atmosphere, practical desk, papers, tablet, simple navy scrubs, realistic Japanese clinic interior. The mood suggests daytime work ending around 5:30 pm without showing readable clock details. Natural shadows, muted warm tones, no dramatic smile, no staged corporate pose, healthcare recruitment photography.
```

ネガティブ要素:

```text
avoid: night shift scene, dark hospital corridor, glowing monitors, fake readable clock, perfect showroom office, stock photo smile, dramatic cinematic blue lighting
```

### 8.5 成田の地域感

## 8B. Midjourney用 イラスト生成プロンプト

現LPは、`https://zaitaku-en.com/` のような柔らかい在宅医療サイトの印象に寄せる。イラストは写真の代替ではなく、ページ全体の空気を作る主役として使う。

### 保存フォルダと命名ルール

Midjourneyで生成したイラストは、実写写真と混ざらないように以下へ保存する。

```text
assets/illustrations/
```

命名形式:

```text
mj-{section}-{purpose}-{ratio}.{ext}
```

ルール:

- 小文字英数字とハイフンのみ
- LPで使う最終画像は `.webp`
- Midjourneyから出した原本や候補は `assets/illustrations/source/` に保存
- 文字入り画像は作らない。見出しやCTAはHTML側で載せる
- 同じ用途で複数案を残す場合、source側だけ `-v01`、`-v02` を付ける

最終ファイル:

| ファイル名 | 用途 | 比率 |
| --- | --- | --- |
| `mj-hero-homevisit-4x3.webp` | ファーストビュー。医師・看護師・診療車・住宅街 | 4:3 |
| `mj-work-visit-bag-4x3.webp` | 仕事紹介。訪問バッグ、物品、準備 | 4:3 |
| `mj-training-meeting-4x3.webp` | 教育。医師と看護師の相談・予定確認 | 4:3 |
| `mj-area-clinic-car-16x9.webp` | 地域感。成田周辺の住宅街と診療車 | 16:9 |
| `mj-contact-chat-1x1.webp` | 最終CTA。スマホ相談、吹き出し、訪問バッグ | 1:1 |
| `mj-ad-nurse-4x5.webp` | 広告用。看護師、訪問バッグ、診療車 | 4:5 |

### 共通方針

- 手描き感のある医療・在宅・地域のイラスト
- 丸く、やさしい線。ただし幼児向けや漫画調に寄せすぎない
- 生成り、ミントグリーン、淡いオレンジ、やわらかい黄色
- 白背景にそのまま載せられる余白
- 日本の在宅医療、成田周辺の住宅街を想起させる落ち着いた雰囲気
- 医療行為中ではなく、訪問前、移動、相談、準備のシーン

避けるもの:

- 未来的な医療、SF、3Dレンダー、光沢の強いベクター
- 青紫グラデーション、硬いビジネスイラスト
- 完璧な左右対称、AIっぽい過剰な装飾
- 患者の顔が主役になる構図
- 実在しない文字、読めそうな看板、架空ロゴ

Midjourney共通パラメータ例:

```text
--ar 4:3 --v 6 --style raw --s 150
```

柔らかさを強める場合:

```text
--ar 4:3 --v 6 --style raw --s 250
```

### Midjourney生成後の選定基準

採用する画像:

- 余白があり、LPの文字やCTAを邪魔しない
- 線が柔らかく、丸みはあるが幼すぎない
- 医療・在宅・地域の文脈が一目で分かる
- 生成り・ミント・淡いオレンジの色調に合う
- 顔や手が破綻していない

不採用:

- 海外の医療機関に見える
- 美容クリニック、病院施設、未来医療のように見える
- 患者の治療場面が主役になっている
- 実在しない文字やロゴが入っている
- 立体感が強すぎる3D、汎用SaaS風ベクター

## 9. 画像生成後の選定基準

採用する画像:

- 人物が自然で、表情が控えめ
- 小物や空間に現実味がある
- 白すぎず、清潔感と生活感が両立している
- 文字を載せても読める余白がある
- 医療行為の誤表現がない

不採用にする画像:

- 手指、器具、文字、時計、車のロゴが破綻している
- 顔が均一でAI生成っぽい
- 海外の病院や美容クリニックに見える
- 高級ホテル風、未来医療風、サイバー風
- 実在しない病院名や読める偽文字が入っている

補正方針:

- 彩度を下げる
- コントラストを少し上げる
- 粒子を薄く加える
- 青みを抜き、生成りと深緑に寄せる
- 肌を過度に補正しない

## 10. 実装メモ

CTA配置:

- PC: ファーストビュー内、オンコール説明後、条件表後、最終セクション
- スマホ: 下部固定で「LINEで質問」「見学」の2択

計測:

- LINEクリック
- 見学フォームクリック
- 条件表到達
- オンコールセクション到達
- FAQ開閉

ABテスト:

- A案: 未経験から訪問診療
- B案: 夜勤から日勤常勤へ
- C案: 成田で長く働く在宅医療

初期はA案を本命にする。オンコールありでも相談化しやすいのは「医師同行」「未経験でも段階的に開始」の説明が最も効くため。

## 11. LP全体の完成イメージ

このLPは、一般的な医療サイトの白・水色・丸いカード構成ではなく、採用パンフレットと地域医療のドキュメンタリーを合わせた見え方にする。

候補者が見終わった時に持つ印象:

- 仕事の内容が想像できる
- オンコールがあることも事前に分かった
- 医師同行なので訪問未経験でも相談できそう
- 写真に現実味があり、職場の雰囲気をごまかしていない
- いきなり応募ではなく、LINEで条件確認や見学から入れる

最終ゴールは応募数だけではなく、直接相談の母数を作ること。月8件相談を狙うため、ファーストビュー、オンコール説明、LINE導線、写真の実在感を最優先で作る。

## 12. 実写画像の配置計画

スマホで撮影した院長・院内・診療車・訪問バッグの写真は、Geminiで補正してLPに配置する。AIで別物に作り替えるのではなく、実在感を残して明るさ、色、不要物、傾き、肌の自然さを整える。

### HTML上の配置

| 画像 | 配置セクション | ファイル名 | 目的 |
| --- | --- | --- | --- |
| 院長写真 | 院長からのメッセージ | `assets/photos/director.jpg` | 採用の信頼感、人柄 |
| 院内写真 | 働く場所の空気 | `assets/photos/clinic-room.jpg` | 職場の広さ、清潔感、実在感 |
| 診療車写真 | 働く場所の空気 | `assets/photos/clinic-car.jpg` | 訪問診療の具体性 |
| 休憩スペース | 働く場所の空気 | `assets/photos/break-room.jpg` | 少人数職場の雰囲気 |
| 訪問バッグ | 働く場所の空気 | `assets/photos/visit-bag.jpg` | 仕事内容の具体性 |

### Gemini補正の方針

院長写真:

- 上半身、自然な表情、白衣または診療着
- 背景は院内、白壁、窓際など
- 肌補正は弱め。別人に見える変換はしない
- 眼鏡、髪型、輪郭、年齢感は残す

院内写真:

- 広角で撮りすぎて歪んだ場合は補正
- 明るくするが、真っ白に飛ばさない
- 掲示物や個人情報は消す
- 実在しない文字・看板は生成しない

診療車:

- ナンバーはぼかす
- 実在ロゴや車体色は残す
- 背景の余計な車や人物は整理してよい
- 高級車風や広告写真風にしない

訪問バッグ:

- 手元、バッグ、物品を自然に見せる
- 医療行為中の写真にしない
- 血液、注射、処置の強い印象は避ける
