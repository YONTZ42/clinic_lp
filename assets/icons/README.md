# GPT image icons

GPT画像生成で作成した、LP内カード用の小さなアイコンです。

| File | Use |
| --- | --- |
| `icon-home-care.png` | 「医師と一緒に訪問」カード |
| `icon-visit-bag.png` | 「準備から覚えられる」カード |
| `icon-chat-consult.png` | 「見学・相談からOK」カード |

方針:

- CSSだけのアイコンではなく、生成画像のやわらかい質感を使う
- 背景はLPになじむ生成り系
- カード内では `124px` 四方で表示
- 文字は画像内に入れない
