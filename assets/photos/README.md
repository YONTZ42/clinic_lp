# LP photo replacements

Geminiなどで補正した実在写真を、以下のファイル名でこのフォルダに置くとLPへ反映されます。

| File | Use |
| --- | --- |
| `director.jpg` | 院長からのメッセージ。院長の上半身写真 |
| `clinic-room.jpg` | 働く場所の空気。院内、診察室、受付、作業スペース |
| `clinic-car.jpg` | 働く場所の空気。診療車、出発前の雰囲気 |
| `visit-bag.jpg` | 働く場所の空気。訪問バッグ、血圧計、物品 |

現在のLPは、ファーストビューや教育セクションの大きなビジュアルを手描き調SVGイラストで構成しています。Midjourneyで生成したイラストに差し替える場合は、HTML内の該当SVGを画像タグまたは背景画像に置き換えてください。プロンプトは `docs/narita-home-clinic-lp-design.md` にまとめています。

推奨:

- 横長写真は1600px以上、縦長写真は1200px以上
- 人物の肌補正は弱め
- 青白さを抑え、生成り・深緑・赤土色に寄せる
- 偽の文字、患者の顔、実態と異なる医療行為が写る写真は使わない
